# easymock
easy-mock-docker

## docker-compose up

run docker:

```shell
docker-compose up -d
```

`docker-compose logs -f` check docker.

or 

visit [http://127.0.0.1:7300](http://127.0.0.1:7300)

> Please note that replacing IP.

## Easy Mock CLI 

install `Easy Mock CLI `:

```shell
npm install -g easy-mock-cli
```

edit `.easymockrc.js`.

generate api files:

```
easymock init .
```
